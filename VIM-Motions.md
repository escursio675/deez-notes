#general

- `Ctrl + z` - Open fuzzy find
- `Space gr` - Live grep
- `Ctrl + v` - Open new file in new vertical window
- `Ctrl + w <h, j, k, l>` - navigate to file in split view
- `gf` - while cursor is on a path, go to that file in new buffer
- `Ctrl + o` - go to previous file, ie, "undo a jump"
- `<line number> G` - jump to 'line number'
- `*` - highlight all instances of a word under the cursor

# Neo-Tree
- `Ctrl + b` - toggle neotree left, `q` to quit
- `a dir/file.extension` - with the cursor in the filesystem pane, add a new file


# Smear-Cursor
- `:SmearCursorToggle` - add or remove smear-cursor effect
